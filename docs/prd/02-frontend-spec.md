# Front-end Specification: ENEXT Compliance Orchestrator

## 1. Brand Identity: "Enext Signature"
Strict adherence to Enext's official color palette and visual aesthetics for B2B SaaS.

### Core Palette
- **Enext Purple:** `#5E48E8` (Main buttons, service cards, active states).
- **Enext Midnight:** `#0D1117` (Page backgrounds, dark mode sidebar).
- **Enext Orange/Gold:** `#F59E0B` (Keywords, Compliance Score, critical borders).

### Aesthetics
- **High-Contrast Tech:** Purple to deep blue gradients.
- **Hierarchy:** Bold typography for future-oriented data presentation.
- **Modernity:** Glassmorphism in modals and subtle glow effects.

## 2. Corporate Dashboard (C-Level & DPO)
- **Central Widget:** Compliance Score Dial (Animated speedo in Enext Orange).
- **KPI Cards:** White text, highlighted numbers in Orange, subtle purple line charts.
- **Side Menu:** Soft gray text, illuminating white/purple on hover.

## 3. Final User Portal (Invisible Portal)
- **Biometric Capture:** Scanning circle with Enext Purple gradient.
- **Cloud Signature:** Action button in Orange for maximum visibility.
- **Animations:** `animate-scan`, `animate-fade-in-up` for premium feedback.
